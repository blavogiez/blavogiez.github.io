# Portfolio - Baptiste Lavogiez

Portfolio moderne et responsive en HTML/CSS/JS vanilla.

## Installation
1. Cloner le repo
2. Modifier `settings.js` avec vos infos
3. Ouvrir `index.html` dans un navigateur

## Structure
- `settings.js` : Configuration (projets, infos perso)  
- `portfolio.js` : Logique JavaScript modulaire
- `main.css` : Styles avec variables CSS
- `index.html` : Page principale

Système multilingue FR/EN, navigation par flèches, galerie auto-rotative.
